# AI Review Detector

A Machine Learning web application that detects whether a product review is **AI-generated (fake)** or **genuinely written by a human**, using Natural Language Processing (NLP).

## About The Project

Online platforms are flooded with fake AI-generated reviews that are designed to look authentic. This project tackles that problem using a trained ML model that learns the difference between computer-generated and original reviews.

The model is trained on 40,000+ real-world product reviews and achieves **88.62% accuracy** on the test set.

> **Note:** The model is trained on a specific dataset and may not always be accurate on all types of reviews. It is best at detecting subtle AI-generated reviews that match the patterns in its training data. Results should not be treated as definitive.

---

## Features

- Detects AI-generated fake reviews using NLP and Machine Learning
- Rule-based filter for catching obvious spam (excessive caps, punctuation)
- Confidence score shown with every prediction
- Word and character count of the input review
- Clean and minimal UI built with Streamlit
- Expandable section to view preprocessed review text

## How It Works

1. **Rule-based filter** — checks for obvious spam signals like excessive caps or exclamation marks
2. **Text cleaning** — removes punctuation, numbers, and extra whitespace
3. **TF-IDF Vectorization** — converts cleaned text into numerical features using unigrams and bigrams
4. **Logistic Regression** — predicts whether the review is fake (CG) or genuine (OR)
5. **Confidence score** — shows how certain the model is about its prediction

## Run Locally

### 1. Clone the repository
```bash
git clone https://github.com/JeanStephanie/ai-review-detector.git
cd ai-review-detector
```

### 2. Create and activate virtual environment
```bash
python -m venv venv
venv\Scripts\activate        # Windows
source venv/bin/activate     # Mac/Linux
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Download the dataset
Download the dataset from Kaggle:
[Fake and Real Product Reviews Dataset](https://www.kaggle.com/datasets/mexwell/fake-reviews-dataset)

Place the downloaded CSV file in the `data/` folder 

### 5. Train the model
```bash
python train.py
```
This generates `model.pkl`, `vectorizer.pkl` and `classes.pkl` inside the `model/` folder.

### 6. Run the app
```bash
streamlit run app.py
```

## ⚠️ Limitations

- Reviews that look very different from the training data may be misclassified
- The rule-based filter only catches obvious spam — not all fake reviews are obvious
